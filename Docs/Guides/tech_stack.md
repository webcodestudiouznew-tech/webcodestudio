# Технологический стек

- **Core:** Next.js 16 + TypeScript
- **UI:** Tailwind CSS v4 + shadcn/ui (New York)
- **i18n:** next-intl v4.4 (RU/UZ/EN с локализованными роутами)
- **Routing for i18n:** для новых мультиязычных страниц использовать App Router structure, в которой локаль может задаваться на уровне root layout без перевода локализованных страниц в dynamic rendering
- **Forms:** react-hook-form + zod
- **Quality:** ESLint - последняя версия
- **Package:** npm
