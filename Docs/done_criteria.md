# Definition Of Done

## Общие критерии
- Проект собирается без ошибок.
- ESLint проходит без ошибок.
- Нет критических визуальных дефектов на desktop и mobile.
- Все основные секции лендинга реализованы.

## Контент и UX
- Hero ясно доносит оффер за первые секунды просмотра.
- Все CTA понятны и визуально согласованы.
- Нет фейковых отзывов, кейсов или неподтверждённых цифр.
- Временные заглушки явно помечены как временные.

## i18n
- Реализованы RU / UZ / EN.
- UI-строки не захардкожены в компонентах.
- Локализация не ломает layout.
- Локализованные route работают корректно.

## Формы и конверсия
- Форма заявки валидируется на клиенте.
- Ошибки формы понятны пользователю.
- Отправка формы работает предсказуемо.
- CTA в Telegram / WhatsApp доступны в ключевых местах.

## SEO
- Заполнены title и description.
- Настроены Open Graph данные.
- Есть базовая semantic structure.
- Подготовлены canonical / alternate links для локалей.

## Качество реализации
- Компоненты разбиты по ответственности.
- Нет дублирования базовых UI-паттернов без необходимости.
- Контент отделён от представления.
- Код понятен для дальнейшего расширения.

## Минимальная проверка перед приёмкой
- `npm run lint`
- `npm run build`
- Ручная проверка в браузере
- Проверка mobile viewport
- Проверка всех CTA и формы
