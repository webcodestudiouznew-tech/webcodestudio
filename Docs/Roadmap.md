- Проект: WebCode
- Тип: лендинг веб-студии

## Цель первой версии
Собрать продающий мультиязычный лендинг для веб-студии WebCode с акцентом на оффер:
сайт под ключ для сервисного и экспертного бизнеса в Узбекистане + CRM для заявок + Telegram/WhatsApp + домен/хостинг + поддержка.
