## Цель первой версии
Собрать продающий мультиязычный лендинг для веб-студии WebCode с акцентом на оффер:
сайт под ключ для локального бизнеса в Узбекистане + CRM в Notion + Telegram/WhatsApp + домен/хостинг + поддержка.
